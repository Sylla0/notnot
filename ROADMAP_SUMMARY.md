# NotNot 개발 로드맵 요약

## 🎯 Phase별 핵심 목표

```
Phase 1 (2-3주): 안정화 및 최적화
├── 코드 최적화 (번들 크기 50% ↓)
├── 버그 수정 (키보드 단축키)
└── UI/UX 개선 (다크 모드)

Phase 2 (4-5주): 핵심 기능 확장
├── 비디오 전사 🎬
├── AI 요약 🤖
└── 고급 편집기 ✏️

Phase 3 (3-4주): 플랫폼 확장
├── Vimeo 지원
├── Coursera 지원
└── Udemy 지원

Phase 4 (5-6주): 클라우드 & 협업
├── 클라우드 동기화 ☁️
├── 실시간 협업 👥
└── Notion/Obsidian 연동

Phase 5 (4-5주): AI 고급 기능
├── 퀴즈 생성 📝
├── 학습 경로 추천 🗺️
└── 음성 기능 🎤

Phase 6 (6-8주): 수익화
├── 프리미엄 플랜 💎
├── 팀/기업용 🏢
└── API/SDK 🔧
```

## 📊 현재 상태 vs 목표

### 현재 구현 완료 ✅
- YouTube 전용 지원
- 로컬 저장소 (IndexedDB)
- 기본 노트 작성
- 스크린샷 캡처
- 마크다운 내보내기

### 6개월 후 목표 🎯
- 4대 플랫폼 지원
- 클라우드 동기화
- AI 기반 학습 도우미
- 실시간 협업
- 수익 모델 확립

```

프로 플랜 ($9.99/월)

팀 플랜 ($19.99/사용자/월)
├── 프로 기능 전체

```
